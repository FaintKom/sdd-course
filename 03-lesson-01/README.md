## Порядок проведения
1. До урока: разослать pre-work, подготовить кейсы/репо/снапшот-ветку (см. facilitator-prep).
2. На уроке: вести по `lesson-plan-w1.md` (environment doctor → концепт → кейсы → карта → выбор темы).
3. Результат урока у студента: заполненная карта доверия/контроля + валидная тема сквозного проекта.
4. ДЗ к W2: дочитать примеры спек, описать задачу 3–5 предложениями.
