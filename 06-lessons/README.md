## Шаблон плана (построен на скилле «Образовательный опыт», С. Смыслова)
Каждый план собран по одной методической рамке — не «лекция + практика», а спроектированный опыт:

- **ОР (ABCD)** — измеримый результат недели (audience · behavior · condition · degree).
- **Вектор опыта** — аффективная цель: какое переживание/сдвиг установки (целеполагание как вектор, ch03).
- **Конструктивная согласованность** — связка ОР ↔ учебная стратегия ↔ метод оценки (Биггс, ch02).
- **Run-of-show с разметкой по циклу Колба** — каждый блок помечен модусом (CE / AC / RO / AE); вход
  не жёстко с CE — студент входит через привычную стратегию (ch03).
- **Нагрузка по Свеллеру** — intrinsic / extraneous / germane + конкретная стратегия снижения
  (рабочий пример, fading, модальность, убрать избыточное); нагрузка переведена во время (ch05).
- **Рефлексия по Гиббсу** — короткий цикл (описание→чувство→анализ→вывод→план) в каждом уроке как
  мост опыт→знание (ch03).
- **Эмоциональная динамика (Пекрун)** — доминирующая эмоция + тип + контрмера; преодолеваемая
  фрустрация, без пиков; активизирующее vs расслабляющее удовольствие (ch05).
- **Мотивационная поддержка** — ценность / ощущение успеха / прозрачность (ch05).
- **Тиры T1/T2/T3** — обратная сторона экспертности (Kalyuga): опора убывает с ростом экспертности.

## Сквозные принципы (общие со всем курсом)
- **Единый каркас:** environment doctor → концепт (AC) → worked example (CE) → своя практика (AE) →
  разбор + рефлексия (RO) → выдача артефакта.
- **Сквозной проект** — каждая неделя = его шаг; артефакт недели накопительно к защите W8.
- **Психобезопасность (CSCI)** — blameless, заданные в W1, держатся весь курс (особенно W5, W7).
- **Две ямы:** W2 (спека трудна) и W7 (чужой легаси, главный риск отвала) — с контрмерами в планах.
