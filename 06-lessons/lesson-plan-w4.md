# План урока W4 — Постановка задач AI-агенту

> **Статус: драфт к согласованию с экспертом** (тайминг, материалы, worked examples).
> Тип 4C/ID: LT. ≈2 ч, live. Артефакт — agent plan + PR по спеке. Intrinsic med, germane high.
> Спроектирован по скиллу «Образовательный опыт»: цикл Колба + Гиббс, нагрузка по Свеллеру,
> эмоции по Пекруну, мотивационная поддержка, чек-лист конструктивной согласованности.

## Образовательный результат (ABCD)
- **A** студент · **B** ставит задачу агенту в ограниченном scope и ведёт по плану с контрольными точками ·
  **C** на атомарной задаче в репо · **D** diff/PR соответствует спеке; лишние изменения отклонены.

## Вектор опыта (аффективная цель)
Студент **переживает позицию ведущего**: «я не нажимаю кнопку и жду — я веду, проверяю на чекпоинтах, отклоняю лишнее». Здесь спека (W2) и контекст (W3) впервые срабатывают вместе — момент «оно работает как система».

## Конструктивная согласованность (чек-лист)
ОР (вести агента) ↔ **стратегия** (рабочий пример петли + практика на своей задаче, Лорильяр: практика) ↔ **оценка** (критериальная: diff по спеке, лишнее отклонено). Смысл — свой шаг проекта.

## Артефакт
Agent plan (шаги, границы, контрольные точки) + PR, соответствующий спеке.

## Pre-work
Готовая спека (W2) + контекст (W3) на текущий шаг проекта. *(Эффект взаимосвязанности — опираемся на готовые артефакты.)*

## Подготовка фасилитатора
Шаблон agent plan; демонстрация agent mode; снапшот-ветка `w4`; пример прогона с выходом за scope — для разбора коррекции.

## Поминутный план (≈120 мин) — разметка по циклу Колба + нагрузка

| Время | Блок | Колб | Что происходит | Нагрузка (стратегия Свеллера) |
|---|---|---|---|---|
| 0:00–0:15 | Environment doctor | — | добор окружения; сломан → ветка `w4` | extraneous ↓ |
| 0:15–0:35 | Концепт: вести, не запускать | **AC** | план, scope, контрольные точки | intrinsic med; всё важное вместе |
| 0:35–1:10 | Worked example: веду агента вживую | **CE (vicarious)** | план→прогон→промежуточный diff→коррекция→отклонение лишнего; T1 повторяют | **рабочий пример** |
| 1:10–1:40 | Своя практика: ставлю в scope, веду | **AE** | эксперт помогает резать scope, ловить выход за рамки | germane high (цикл план→diff→коррекция) |
| 1:40–1:52 | Где агент вышел за scope + рефлексия Гиббса | **RO** | показ «лишних правок» и как отклонять | germane |
| 1:52–2:00 | Закрытие | — | критерий артефакта + pre-work W5 | — |

*Вход: T1 — от рабочего примера (CE), T2/T3 — сразу ведут свою задачу (AE).*

## Рефлексия (Гиббс, кратко, в блоке 1:40)
Описание (как вёл агента) → чувство (где терял контроль?) → анализ (на каком шаге агент ушёл за scope) → вывод (контрольные точки удерживают) → план (где поставлю чекпоинт в следующий раз).

## Эмоциональная динамика (Пекрун)
Доминанта — **ощущение контроля (эмоция достижения +)**, восстановление после ямы #1. Риск — эйфория «агент всё может» → завышенное доверие; контрмера — обязательный разбор выхода за scope (отклонение лишнего = норма, не придирка).

## Мотивационная поддержка
**Ценность:** управление AI-агентами = строка резюме. **Успех:** первый PR, соответствующий спеке. **Прозрачность:** шаг 4/8, проект ощутимо движется.

## Проверка (формирующая, критериальная)
diff/PR соответствует спеке; **лишние изменения отклонены** (видно в истории коррекций); виден план + контрольные точки, не только генерация.

## ДЗ к W5
Провести задачу агентом по плану до PR; прочитать чек-лист ревью (`../02-materials/review-checklist-ai-code.md`) — вход в W5. «Принято»: PR по спеке. ~3 ч.

## Тиры (обратная сторона экспертности)
- **T1:** по шаблону плана, мелкий scope. · **T2:** свой план, фича+багфикс. · **T3:** prompt chaining / subagent-разбиение, чекпоинты на риск.

## Риски урока
- Разнобой инструментов → один основной агент на поток. · Раздутый scope → атомарная задача, режем вместе. · Принимает всё от агента → тренируем отклонение лишнего.
