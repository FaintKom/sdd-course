# Структура курса: модули → уроки → нагрузка

> Скелет для SME. Формат: **blended**. Модуль = неделя. Внутри модуля единый ритм:
> асинхронные уроки до воркшопа (теория 10–20 мин + мини-практика) → **live-воркшоп 2 ч** (ядро) →
> ДЗ = шаг сквозного проекта → рефлексия. Детальные скелеты уроков — `01-module-w1.md` … `08-module-w8.md`.
>
> Обозначения 4C/ID: **LT** учебная задача (целостная) · **SI** поддерживающая информация (ментальные
> модели, «как думает эксперт») · **PI** процедурная информация (инструкции just-in-time) · **PP** частичная
> практика (дрилл рутинного навыка). Канон программы — `../program-and-outcomes.md` (не менялся).

## Ритм модуля (единый каркас — снижает extraneous-нагрузку повторяемостью)

| Слот | Тип урока | 4C/ID | Когда | Хронометраж студента |
|---|---|---|---|---|
| x.1 | Концепт (лонгрид + скрины/скринкаст + квиз) | SI | до воркшопа | 15–20 мин |
| x.2 | Процедура/дрилл (разбор + мини-практика) | PI или PP | до воркшопа | 20–30 мин |
| x.3 | **Live-воркшоп** (worked example → своя практика) | LT | синхронно | 120 мин |
| x.4 | Практика проекта (ДЗ = шаг сквозного проекта) | LT | после | 1–4 ч |
| x.5 | Рефлексия + мост к следующей неделе | — (Гиббс) | после | 10–15 мин |

Принцип pre-training (Свеллер/Мейер): концепты выносим в асинхрон **до** воркшопа → на live меньше
intrinsic-нагрузки, больше времени на практику. Воркшопы уже спроектированы (`../03-lesson-01/`,
`../06-lessons/`) — скелет их не дублирует, а достраивает асинхронную обвязку.

## Шаблон скелета урока (что получает SME в файлах модулей)

Каждый урок в `0N-module-wN.md` описан так:
- **Цель урока (микро-ОР)** — что студент умеет после урока (наблюдаемый глагол).
- **4C/ID-компонент** — SI / PI / PP / LT и почему.
- **Тезисный контент** — 5–10 тезисов, порядок подачи, примеры/антипримеры (SME раскрывает).
- **Мини-практика / задачи** — формулировка, входные материалы, критерий правильности, авто/ручная проверка.
- **Материалы к производству** — что написать/сверстать/подготовить (лонгрид, скринкаст, шаблон, репо, diff).
- **Связи** — на что опирается (урок назад) и что готовит (урок вперёд).
- **Тиры T1/T2/T3** — где убывает опора.
- **[SME-slot]** — места, которые заполняет эксперт (примеры из практики, цифры, истории провалов).
