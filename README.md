Точка входа. 8 недель, живые воркшопы, ЦА junior+ → senior/lead. Канон — гугл-таблица курса.
Этот README = карта папки + статус готовности к старту разработки + бэклог.

## С чего начать (роли)
- **Эксперт / SME** → `sme-kickoff-kit.html` (бенчмарк, структура, примеры заданий, вопросы к тебе §9).
- **Методист / продюсер** → `program-and-outcomes.md` (программа + ОР) + `sdd-course-design.html` (полный LXD-дизайн).
- **Фасилитатор первого урока** → `03-lesson-01/README.md`.

## Вывод
Для **старта работы** (встреча с SME + начало разработки) есть всё: программа, ОР, дизайн на основе
бенчмарка, материалы-прототипы, первый урок. Блокер до сборки контента — **решения SME** (§9 kickoff-kit).
Производственный код (репо-шаблоны, легаси, CI) собирается после подтверждения стека.
